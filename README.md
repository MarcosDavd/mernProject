# Proyecto MERN - Autenticación y Autorización de Usuarios

Este proyecto es una aplicación web desarrollada con el stack MERN (MongoDB, Express.js, React y Node.js) para implementar autenticación y autorización de usuarios.

## ¿Qué incluye?

- Backend con Express.js y MongoDB
- Registro y login de usuarios
- Autenticación basada en JWT
- Manejo de sesiones y cierre de sesión
- Recuperación de contraseña mediante correo electrónico
- Frontend con React y React Router
- Pantallas básicas de login y home

## Estructura del proyecto

- backend/: lógica del servidor, modelos, rutas, controladores y configuración de base de datos
- frontend/: aplicación React con Vite, rutas y componentes de interfaz

## Tecnologías utilizadas

### Backend
- Node.js
- Express.js
- MongoDB + Mongoose
- JWT
- bcryptjs
- Nodemailer

### Frontend
- React
- Vite
- React Router DOM
- Axios
